# Zenn 自動記事生成プロジェクト

## プロジェクト概要
Claude Code を使ってAI・データサイエンス系のZenn記事を生成し、GitHubにpushするプロジェクト。

## 記事生成の手順

「新しい記事を1本生成してpushしてください」と頼まれたら以下を実行する:

1. **「執筆済み記事」セクションを確認し、重複しないトピックを選ぶ**
2. 下記カテゴリ・トピックリストから1つ選ぶ
3. `articles/` に Zennフォーマットの `.md` ファイルを作成する
4. **記事ファイルを1本ずつ別のcommitでpushする**
   ```
   git add articles/{ファイル名}.md && git commit -m "docs: タイトル" && git push
   ```
   ⚠️ Zennの無料プランは1日2本までしか新規デプロイできない。1日1本ペースが安全。
5. **「執筆済み記事」セクションに書いた記事を追記する**
6. git push が完了したら必ず以下を実行してX投稿用テキストを表示する:
   ```
   python scripts/post_to_x.py articles/{作成したファイル名}.md
   ```
   表示されたテキストをそのままユーザーに見せる（ユーザーがXにコピペする）

## Zenn 記事フォーマット

ファイル名: `YYYYMMDD-{英語スラッグ}.md`（例: `20250604-data-scientist-growth.md`）

```markdown
---
title: "記事タイトル（日本語OK）"
emoji: "📊"
type: "tech"  # tech: 技術記事 / idea: 意見・考察
topics: ["python", "datascience", "statistics"]  # 最大5個
published: true
---

## はじめに
（背景・なぜこれが重要か）

## 本題
（複数セクション、コード例を含む）

## まとめ
（要点整理・次のステップ）
```

### typeの使い分け
- `tech`: 実装・ハンズオン系、コードが主体
- `idea`: キャリア・考察・ニュース解説系

### 執筆ルール
- 日本語で執筆
- 対象読者: Python中級者（1〜3年目エンジニア）
- 文字数: 3000〜5000文字
- techの場合はPythonコード例を必ず含める
